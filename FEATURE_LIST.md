# 天穹 (sty-visit) 产品功能清单描述文档

本文档详细列出了 **天穹 (sty-visit v1.0 Go重构版)** 的全量产品功能，旨在为用户提供清晰的功能导览与业务规格参考。

---

## 一、 接入与单文件极简部署 (Deployment & Access)

### 1.1 跨架构零依赖发行包
*   **功能名称**：Universal 智能多架构启动
*   **功能描述**：项目彻底摒弃 Java 与 Docker 依赖。通过单二进制文件封装，结合 `start.sh` 脚本可智能在 x86_64 与 ARM64 环境下秒级拉起服务。
*   **业务价值**：为军工、政务等特殊内网及信创国产服务器提供开箱即用的交付体验。

### 1.2 隐身与认证隔离
*   **功能名称**：Stealth Mode (深度伪装面板)
*   **功能描述**：管理后台设有“H5管理面板开关”，一键关闭后，任何 Web 访问请求将强制伪装为原生的 404 Not Found 界面，抵御端口爆破与探测扫描。

---

## 二、 服务器看板与监控 (Dashboard & Monitoring)

### 2.1 实时性能概览
*   **功能描述**：在首页看板实时展示服务器运行时间 (Uptime) 及关键系统指标（CPU负载、内存等）。

### 2.2 磁盘健康扫描
*   **功能描述**：自动识别服务器挂载点，并以进度条直观展示总容量、已用空间及剩余百分比。

---

## 三、 高性能 SSH 会话 (SSH Session)

### 3.1 独立配置与快捷书签
*   **功能描述**：提供独立的 SSH 连接页。支持将高频服务器存为标签/书签，点击即刻回填账号密码完成瞬连。

### 3.2 ANSI 全彩终端与过载熔断
*   **功能描述**：底层自研 SSH 引擎完整支持 ANSI 16/256 色及各种复杂命令（`top`, `htop`, `vim` 等）。
*   **系统熔断机制**：后端设定 `max_sessions` 的硬性安全上限（默认 50），防范超量会话导致目标机或自身服务崩溃 OOM。

---

## 四、 物理级文件管理 (SFTP Explorer)

### 4.1 SFTP 全功能漫游与交互
*   **功能描述**：支持全盘图形化漫游，支持多层级无感跳转。支持文件的在线编辑（编辑完自动保存回源服务器）、重命名、删除操作。

### 4.2 双端传输管理中心
*   **功能描述**：下载/上传任务后台异步化，统一收口至“传输”模块，提供上传及下载进度百分比追踪。

---

## 五、 系统审计与指令流 (Audit & Snippets)

### 5.1 全局行为审计
*   **功能描述**：静默监控所有登录、控制动作，将所有终端记录沉淀于 `/audit` 目录下，并设置自动超期（如 7 天）销毁，以释放磁盘。

### 5.2 共享命令片段库
*   **功能描述**：支持预设常用自动化指令（Snippet），操作人员点击后可直接下发至当前激活终端。

---

## 六、 批量集控执行 (Batch Operations)

### 6.1 集群并发执行引擎
*   **功能名称**：资产多节点批量发号施令
*   **功能描述**：通过 `nodes` 池或临时圈选服务器，将预设的 shell 脚本（如巡检、重启）一键分发至上百台集群机器，系统将并发执行并聚合返回所有成功或失败结果。
*   **业务价值**：将单机堡垒机跃升为集群运维中控台。
