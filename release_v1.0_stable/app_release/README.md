# 天穹 (sty-visit v1.0) - Android 客户端使用向导

欢迎来到天穹移动端。作为一款端云一体的远程运维堡垒机，Android 端不仅是一个 SSH 客户端，更是整个服务中枢的“物理密钥”与“监视器”。

## 📦 包含的文件
- `sty-visit.apk`：已完成 V1/V2 签名的安卓标准安装包。

---

## 🛠️ 1. 安装与鉴权绑定
1. **安装 APK**：请将 `.apk` 文件发送至您的 Android 设备，并允许“安装来自未知来源的应用”。
2. **服务器绑定**：打开 App，在首次启动的初始化界面中输入：
   - **服务器地址**：填写您部署后端服务的 IP 和端口（例如：`http://192.168.1.100:8080`）
   - **访问密钥**：填写后端 `application.yml` 中对应的 `styvisit.admin-password`。
3. **极简自适应**：后续再次打开 App 时，IP 地址框会自动隐藏，仅需输入密码即可秒级进入系统。

---

## 🛡️ 2. 核心移动端安全特性 (必读)

为了极致的安全考量，App 端被赋予了最高的管理权限，请务必了解以下两项功能：

- **网页管理面板开关 (物理隐身)**
  您可以在 App 的“监控面板”中找到 **“H5 面板准入”** 开关。当您将其关闭时，外部任何通过浏览器访问 8080 端口的请求，都会被后端拦截并直接扔回一个无特征的 `404 Not Found` 伪装页面。这能帮您彻底规避僵尸网络的端口扫描！

- **全域物理擦除 (紧急撤离)**
  当遇到手机可能脱离控制、或怀疑后端被非授权访问的紧急时刻，您可以点击 App 右上角红色的 **【强制退出并擦除】** 按钮。
  *此时，不仅您的手机应用缓存会被全部清空，App 还会向服务器发送销毁指令，擦除后端的所有配置字典、服务器密码簿以及历史记录文件，确保物理层面的绝对安全。*
